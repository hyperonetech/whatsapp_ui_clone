### Installation

 requires [Flutter](https://flutter.dev/)  to run.

Install the dependencies and devDependencies and run app.

```sh
$ git  clone https://github.com/hyperonetech/whatsapp_ui_clone.git
$ cd whtasapp_ui_clone_git
$ flutter pub get
$ flutter run
```
